## ⚙️ Setup Guide (CLI Method)

Follow these steps to run Qwen3 Coder Next locally:

### ✅ Step 1: Install Node.js
Make sure Node.js (v20+) is installed:

```
node -v
```

Download: https://nodejs.org

---

### ✅ Step 2: Install Qwen CLI

```
npm install -g @qwen-code/qwen-code
```

Verify:

```
qwen --version
```

---

### ✅ Step 3: Start Qwen CLI

```
qwen
```

---

### ✅ Step 4: Login (OAuth - FREE)

Run inside CLI:

/auth

or simply press **Enter** if OAuth login option is shown.

- Select **Qwen OAuth Login**  
- A browser window will open  
- Sign in using your Qwen account  
- Authentication completes automatically ✅  

👉 You now get FREE usage (up to ~2000 requests/day)

---

### ✅ Step 5: Generate Game

Example prompt:

```
Create a snake game in Python using pygame
```

---

## 🎮 Run the Game

### Install dependencies:

```
pip install -r requirements.txt
```

If you face issues:

```
pip install pygame-ce
```

---

### Run:

```
python snake_game.py
```
